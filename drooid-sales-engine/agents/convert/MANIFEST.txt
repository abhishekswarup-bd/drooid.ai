AGENT FILES (8 total)
====================

10-pre-call-researcher.js (9.3K)
  Agent ID: agent-10
  Function: Deep research and briefing before sales meetings
  Input: lead_id, contact_id, meeting_details
  Output: Structured briefing (exec_summary, talking_points, questions, objections)
  Storage: pipeline.notes + content (pre_call_brief)
  Approval Required: NO
  Gemini Temperature: 0.3 (research focused)
  Timeout: 30s

11-proposal-writer.js (11K)
  Agent ID: agent-11
  Function: Custom proposal generation after qualified meetings
  Input: lead_id, contact_id, meeting_notes, requirements
  Output: Full markdown proposal + deal_value + win_probability
  Storage: content (proposal) + approvals (pending CEO)
  Approval Required: YES
  Gemini Temperature: 0.4 (creative but focused)
  Timeout: 45s

12-pipeline-tracker.js (12K)
  Agent ID: agent-12
  Function: Daily pipeline health monitoring and risk flagging
  Input: All active deals from pipeline table
  Output: Health report (healthy/at-risk/stale) + recommendations
  Storage: agent_metrics (pipeline_health) + tasks (actions)
  Approval Required: NO
  Gemini Temperature: 0.3 (analytical)
  Timeout: 40s

13-nurture-agent.js (11K)
  Agent ID: agent-13
  Function: Value-focused nurture for warm prospects
  Input: Contacts in nurture stage
  Output: Personalized nurture touchpoint per contact
  Storage: outreach (message_type: nurture) + scheduling
  Approval Required: YES
  Gemini Temperature: 0.5 (personalized)
  Timeout: 20s per contact

14-competitive-intel.js (10K)
  Agent ID: agent-14
  Function: Competitive battlecard creation and caching
  Input: competitor_name, optional: deal_id
  Output: Competitive battlecard (strengths, weaknesses, differentiation)
  Storage: content (competitive_intel) + deal_competitive_intel links
  Approval Required: NO
  Gemini Temperature: 0.3 (analytical)
  Timeout: 35s (or 2s if cached)

15-win-loss-analyst.js (12K)
  Agent ID: agent-15
  Function: Analysis of closed deals for patterns and learning
  Input: period_days (default: 90), include_lost flag
  Output: Win/loss report + pattern analysis + recommendations
  Storage: agent_metrics + content (win_loss_analysis)
  Approval Required: NO
  Gemini Temperature: 0.3 (analytical)
  Timeout: 45s

16-referral-generator.js (13K)
  Agent ID: agent-16
  Function: Referral request generation from satisfied customers
  Input: Auto-scans closed_won deals (60 days) + engaged prospects
  Output: Personalized referral requests + talking points
  Storage: outreach (message_type: referral_ask) + approvals (pending)
  Approval Required: YES
  Gemini Temperature: 0.5 (personalized)
  Timeout: 20s per candidate

17-testimonial-collector.js (14K)
  Agent ID: agent-17
  Function: Testimonial and case study collection
  Input: Closed_won deals 30-90 days old
  Output: Testimonial requests + suggested drafts
  Storage: outreach (message_type: testimonial_request) + content (case_study)
  Approval Required: YES
  Gemini Temperature: 0.5 (personalized)
  Timeout: 25s per client
